## 🔧 Quick Start

### Scenario 1: New Project
1. Use `tech-stack-decision.md` → Chọn công nghệ
2. Use `architecture-decision-record.md` → Document major decisions
3. Use `sitemap-structure.md` → Thiết kế IA
4. Use `milestone-roadmap.md` → Lập timeline

### Scenario 2: Feature Planning
1. Use Cursor patterns → Break down tasks
2. Use Antigravity patterns → Create implementation plan
3. Get approval → Start execution

### Scenario 3: Architecture Change
1. Document current state
2. Use ADR template → Evaluate options
3. Create migration plan
4. Update roadmap
