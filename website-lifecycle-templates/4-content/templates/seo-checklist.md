# SEO Optimization Checklist

> **Mục đích**: Đảm bảo website được tối ưu cho search engines

---

## 🔍 On-Page SEO

### Meta Tags
- [ ] **Title tag**: 50-60 characters, include primary keyword
- [ ] **Meta description**: 150-160 characters, compelling copy
- [ ] **H1 tag**: One per page, include primary keyword
- [ ] **H2-H6 tags**: Hierarchical structure, include secondary keywords

### Content
- [ ] **Keyword density**: 1-2% for primary keyword
- [ ] **Keyword placement**: Title, H1, first paragraph, conclusion
- [ ] **Content length**: 800+ words for blog posts
- [ ] **Internal links**: 3-5 relevant links per page
- [ ] **External links**: 1-2 authoritative sources

### Images
- [ ] **Alt text**: Descriptive, include keywords when relevant
- [ ] **File names**: Descriptive (e.g., `blue-running-shoes.jpg`)
- [ ] **Image compression**: < 100KB per image
- [ ] **Lazy loading**: Enabled for below-fold images

### URLs
- [ ] **URL structure**: Short, descriptive, include keywords
- [ ] **Hyphens**: Use hyphens (not underscores) to separate words
- [ ] **Lowercase**: All lowercase letters
- [ ] **No parameters**: Avoid `?id=123` style URLs

---

## 🌐 Technical SEO

### Performance
- [ ] **Page speed**: < 3 seconds load time
- [ ] **Core Web Vitals**: Pass all metrics (LCP, FID, CLS)
- [ ] **Mobile-friendly**: Responsive design
- [ ] **HTTPS**: SSL certificate installed

### Indexing
- [ ] **Sitemap.xml**: Generated and submitted to Google Search Console
- [ ] **Robots.txt**: Configured correctly
- [ ] **Canonical tags**: Set for duplicate content
- [ ] **Structured data**: Schema.org markup (JSON-LD)

### Crawlability
- [ ] **No broken links**: All links working (404 check)
- [ ] **No redirect chains**: Max 1 redirect per URL
- [ ] **XML sitemap**: Updated and accessible

---

## 📱 Local SEO (if applicable)

- [ ] **Google Business Profile**: Claimed and optimized
- [ ] **NAP consistency**: Name, Address, Phone consistent across web
- [ ] **Local keywords**: Include city/region in content
- [ ] **Local schema**: LocalBusiness schema markup

---

## 📊 Monitoring

- [ ] **Google Search Console**: Setup and monitoring
- [ ] **Google Analytics**: Tracking installed
- [ ] **Keyword rankings**: Track top 10 keywords
- [ ] **Backlinks**: Monitor with Ahrefs/SEMrush

---

## 🎯 Priority Actions

| Priority | Action | Impact |
|----------|--------|--------|
| 🔴 High | Fix broken links | Immediate |
| 🔴 High | Optimize page speed | High |
| 🟡 Medium | Add schema markup | Medium |
| 🟢 Low | Build backlinks | Long-term |
