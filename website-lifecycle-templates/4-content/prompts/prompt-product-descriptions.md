> **Source**: NotionAI, Perplexity, Claude Code
> **Use case**: Generate compelling product descriptions for e-commerce

## 🎯 Prompt Template: Product Description

### Basic Product Description

```
Generate a product description for an e-commerce flower shop:

Product: {product_name}
Category: {category}
Price: {price}
Key features: {features}

Requirements:
- Length: 150-200 words
- Tone: Warm, elegant, romantic
- Include: Occasion suggestions, care instructions, delivery info
- SEO keywords: {keywords}
- Call-to-action: Encourage purchase

Format:
1. Opening hook (1-2 sentences)
2. Product details (2-3 sentences)
3. Occasion suggestions (1-2 sentences)
4. Care instructions (1 sentence)
5. Call-to-action (1 sentence)
```

### Example Usage

**Input:**
```
Product: Red Rose Bouquet
Category: Roses
Price: 500,000 VND
Key features: 12 premium red roses, hand-tied, includes vase
Keywords: red roses, romantic gift, anniversary flowers
```

**Output:**
```
Celebrate love with our stunning Red Rose Bouquet, featuring 12 premium 
long-stemmed red roses expertly hand-tied by our master florists. Each 
rose is carefully selected for its vibrant color and perfect bloom, 
creating a timeless expression of romance and passion.

Perfect for anniversaries, Valentine's Day, or simply saying "I love you," 
this classic arrangement arrives beautifully presented with a complimentary 
glass vase. The deep crimson petals and fresh green foliage create an 
elegant display that will brighten any room.

To keep your roses fresh for 7-10 days, trim stems at an angle, change 
water daily, and keep away from direct sunlight.

Order now for same-day delivery in Ho Chi Minh City. Make someone's day 
unforgettable with the timeless beauty of red roses.
```

## 📋 Best Practices

### Do's
✅ Start with benefit, not feature
✅ Use sensory language (colors, scents, textures)
✅ Include specific details (number of stems, size, care time)
✅ Address common objections (delivery, freshness, care)
✅ End with clear call-to-action
✅ Optimize for mobile reading (short paragraphs)

### Don'ts
❌ Generic descriptions ("beautiful flowers")
❌ Keyword stuffing
❌ Overpromising ("will last forever")
❌ Ignoring target audience
❌ Missing practical information (price, delivery, care)
❌ Too technical or too casual
