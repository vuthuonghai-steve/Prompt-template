# Quick Start Guide

> **Mục đích**: Hướng dẫn nhanh sử dụng website-lifecycle-templates
> **Thời gian đọc**: 5 phút

---

## 🚀 Workflow cơ bản

### Bước 1: Chọn Phase
```
Discovery → Planning → Design → Content → Development → Testing → Maintenance
```

### Bước 2: Sử dụng 4 loại tài liệu
```
phase/
├── patterns/     # Best practices
├── prompts/      # AI prompts
├── templates/    # Document templates
└── examples/     # Code examples
```

### Bước 3: Apply vào project
1. Copy templates cần thiết
2. Sử dụng AI prompts để generate content
3. Học patterns và apply
4. Tham khảo examples

---

## 📋 Timeline ví dụ

**Startup MVP (3 tháng)**:
- Discovery: 1 tuần
- Planning: 1 tuần  
- Design: 2 tuần
- Development: 6 tuần
- Testing: 2 tuần

**Enterprise (6-12 tháng)**:
- Discovery: 4 tuần
- Planning: 4 tuần
- Design: 6 tuần
- Development: 16 tuần
- Testing: 4 tuần

---

## 💡 Tips

1. **Không cần follow 100%** - Chọn phù hợp với project
2. **Customize templates** - Adapt cho context của bạn
3. **Sử dụng AI prompts** - Copy vào Claude Code, Cursor
4. **Học từ examples** - Real-world implementations
