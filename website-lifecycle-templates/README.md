# Website Lifecycle Templates

> **Hệ thống template chuẩn cho 7 phases xây dựng website**
> **Nguồn kiến thức**: Trích xuất từ system-prompts-and-models-of-ai-tools
> **Last Updated**: 2026-04-23

---

## 🎯 Mục đích

Cung cấp templates, guidelines, checklists có cấu trúc cho từng giai đoạn phát triển website, dựa trên best practices từ các AI coding tools hàng đầu (Cursor, Claude Code, Windsurf, Replit, Lovable, v.0, Bolt).

---

## 📁 Cấu trúc

```

├── 5-development/         # Phát triển code

└── README.md
```

---

## 🔄 7 Phases

| Phase | Mục tiêu | Đầu ra |
|-------|----------|--------|
| **1. Discovery** | Làm rõ requirements, user stories, scope | Requirements doc, use cases, risk analysis |
| **2. Planning** | Architecture, sitemap, wireframe, tech stack | IA, milestone plan, low-fi wireframes |
| **3. Design** | UI/UX design, components, responsive | High-fi designs, prototype, design system |
| **4. Content** | Copywriting, SEO, metadata, assets | Content guidelines, copy, images, videos |
| **5. Development** | Frontend + Backend implementation | Working application, API, database |
| **6. Testing** | QA, performance, security, deployment | Test reports, production deployment |
| **7. Maintenance** | Bug fixes, optimization, new features | Updates, performance improvements |

---

## 🚀 Cách sử dụng

1. **Chọn phase** đang làm việc
2. **Đọc INDEX.md** trong thư mục phase
3. **Sử dụng templates** phù hợp
4. **Follow checklists** để đảm bảo quality
5. **Tham khảo examples** khi cần
